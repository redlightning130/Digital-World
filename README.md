# Digital World

Digital World is a browser-based exploration game built with HTML and JavaScript.
The player progresses through multiple abstract levels inside a digital system.

## How to Run
Open `title_screen.html` in a web browser.

## Project Structure
- `title_screen.html` – game entry point
- `digital_world.html` – core logic
- `level_one.html` → `level_five.html` – game levels
